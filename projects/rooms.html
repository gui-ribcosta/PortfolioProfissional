Meeting Room System

Concluído

Sistema web completo para gerenciamento de salas de reunião, com controle de participantes por códigos de acesso únicos. Ele permite criar, listar e deletar salas, adicionar participantes, gerar códigos únicos e controlar o acesso de forma prática. Ideal para equipes pequenas, eventos ou reuniões rápidas.
